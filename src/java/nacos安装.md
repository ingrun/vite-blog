1. https://nacos.io/zh-cn/docs/quick-start.html

2. 下载nacos 包   https://github.com/alibaba/nacos/releases

3. 解压安装

   ```bash
   mkdir /data/nacos
   cd /data/nacos
   # 上传文件
   
   tar -zxvf ./nacos-server-2.4.0.1.tar.gz 
   cd nacos/bin
   
   
   ```

4. 运行

   ```bash
   # 单机模式运行
   sh startup.sh -m standalone
   
   ```
