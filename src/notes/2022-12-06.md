## uni.navigateTo  事件监听

```js
toIndex() {
  let _that = this
  const href = '/pages/index/index2' 
  uni.navigateTo({
    url: href + "?rkuid=" + this.rkuid,
    events: {
      // 为指定事件添加一个监听器，获取被打开页面传送到当前页面的数据
      reGetFwlist: function() {
        _that.getFwList()
      }
    }
  })
}
```

被打开的页面调用
```js
const eventChannel = this.getOpenerEventChannel();
eventChannel.emit('reGetFwlist', {data: ''});
```
